A modern and lightweight notification system plugin designed for WordPress directories built with Directorist.

== Description ==

**Directorist Notification System** allows you to send and display instant notifications inside your WordPress directory website.  
It is built for directory-based sites powered by [Directorist](https://directorist.com/), but it can work on any WordPress installation.

This plugin uses a modular architecture with Composer and PSR-4 autoloading, providing clean and scalable code.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install it via the WordPress Plugins menu.
2. Run `composer install` in the plugin root if needed.
3. Activate the plugin through the ‚ÄòPlugins‚Äô screen in WordPress.
